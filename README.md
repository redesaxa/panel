# INSTALL CENTOS WEBPANEL

1. Login vps root
2. ```yum -y install wget```
3. ```yum -y update```
4. ```cd /usr/local/src```
5. ```wget http://centos-webpanel.com/cwp-el7-latest```
6. ```sh cwp-el7-latest```
7. ```reboot```
